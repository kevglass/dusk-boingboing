# Boing Boing!

Boing Boing, a standard jumper game with the added bonus of multiplayer racing!

## How to Play

1) Join the game
2) Left/Right to control your player
3) Avoid spikes, birds and falling off the screen
4) Hit the springs for extra boosts
5) Go faster!
